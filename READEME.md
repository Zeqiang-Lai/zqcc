# ZQCC

The compiler for zqc(a subset of C).

## Usage
```shell
OVERVIEW: zqcc compiler

USAGE: zqcc [options] <input>

OPTIONS:
    -l      Lexer output
    -p      Parser output
    -dir    Output directory
```
